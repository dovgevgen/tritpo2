  «NewsBox» - интерактивный новостной сайт с возможностью выбора новостей в соответствии с персональными предпочтениями. Сайт будет представлять из себя площадку с новостными записями различной тематики и направленности, где пользователь сможет выбрать для себя интересующую тему. Зарегистрированные пользователи могут создавать свои новостные записи, а также комментировать записи других пользователей.
